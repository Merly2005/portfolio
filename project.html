AI-Based Sign Language Interpreter

An intelligent system developed to convert sign language into text and voice using AI technologies. This project uses YOLO and MediaPipe for hand gesture recognition and tracking.

The main objective of this project is to improve communication accessibility for people with hearing and speech impairments through real-time gesture detection.
